# CV Builder

Xây dựng CV chuyên nghiệp với sự hỗ trợ của AI.

## Tính năng

- **Tạo CV chuyên nghiệp**: Format theo mẫu itviec với 8 phần
- **AI phân tích CV**: So sánh CV với công việc ứng tuyển
- **Multi-provider AI**: OpenAI, Anthropic, OpenRouter, LiteLLM, Google AI, VertexAI
- **Quản lý mẫu CV (Admin)**: Upload mẫu PDF, AI phân tích
- **Lưu trữ cloud**: CV được lưu theo tài khoản

## Tech Stack

- Next.js 14 (App Router)
- TypeScript
- TailwindCSS
- Supabase

## Deploy

Deployed at: https://cv-builder.vercel.app

## Setup

1. Clone repo
2. `npm install`
3. Add Supabase env vars
4. `npm run dev`
